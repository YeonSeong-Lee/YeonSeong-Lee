## 👋 About Me

> **사용자의 불편을 코드로 푸는 소프트웨어 엔지니어**입니다.

- 🎓 **42 Seoul** 기반으로 프론트엔드부터 백엔드, AI 응용까지 폭넓게 다룹니다.
- 🚀 아이디어를 **실제 사용자가 쓰는 서비스로 배포·운영**하는 것을 좋아합니다.
  - 📚 42서울 도서관 시스템 **집현전** — **1,000+ 사용자**가 사용하는 서비스를 8인 팀으로 운영
  - 🌐 **sseuregi-king**, **OpenUmbrella** 등 직접 기획·개발·배포
  - 🧩 **Chrome Web Store**에 편의성 확장 프로그램 출시
- 📦 npm 패키지 2종 배포, 🎤 컨퍼런스 발표 및 오픈소스 협업 경험.
- 🧠 새로운 도메인(CV, LLM, 3D 그래픽스)을 빠르게 학습해 프로덕트에 녹입니다.
